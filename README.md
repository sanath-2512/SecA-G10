### 1. Data Cleaning (The "Grunt Work")
- **Handling Missing Data:** usage of `AVERAGEIF` for Item Weight and `MODE` for Outlet Size.
- **Standardizing Labels:** Correcting "LF", "low fat" -> "Low Fat".
- **Sanity Checks:** Removing duplicate records to ensure data integrity.

### 2. Deep-Dive Analysis
- **Pivot Tables:** Used to aggregate sales data by *Item Type*, *Outlet Location*, and *Outlet Type*.
- **Metrics Calculated:** Total Sales, Average Sales per Item, Outlet Performance Index.

### 3. Dashboard Construction
- Integrated slicers for dynamic filtering.
- Designed a color-coded heatmap for regional performance.
